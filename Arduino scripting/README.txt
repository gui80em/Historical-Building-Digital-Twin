IoT_data_adq.ino is the complete script for the microcontroller that does a reading and writes it in the
ThingSpeak server.

mhz19_demo.ino does a test for the CO2 sensor mhz19.
sht3x.ino does a test for the SHT31 temperature and humidity sensor.
tempt6000_demo.ino does a test for the TEMPT6000 light intensity sensor.
esp8266_demo.ino does a test of write in the ThingSpeak server channel.
